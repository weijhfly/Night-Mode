# 夜间助手
夜间助手，可设置网页背景色的浏览器扩展，在浏览器扩展/附加组件中搜索“夜间助手”，即可安装使用。

## 演示   
![Night-Mode](https://weijhfly.github.io/images/unnamed.png)

## 特色
- 十二背景色可选，美观实用；
- 体积小，稳定性、兼容性强；
- 覆盖浏览器广，目前已在火狐、谷歌、360浏览器中上架；
## 安全性：
此扩展源码公开，无任何侵犯隐私问题。
